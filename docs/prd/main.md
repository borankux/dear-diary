## 用户画像
- 主要用户：开发者、Vim/终端重度用户、个人日记爱好者
- 次要用户：本地优先（local-first）数据倡导者

## 核心场景
1. **快速写今天的日记** — 终端敲 `diary`，秒进 Vim
2. **回看历史** — `diary browse` 月历，看哪天写了哪天没写
3. **同一天多次记录** — 早上、中午、晚上各写一段，自动追加时间戳段落
4. **找回内容** — `diary search` 全文搜索历史日记

## 不做的（YAGNI）
- 加密（让 macOS FileVault / iCloud ADP 处理）
- 多用户、云同步（让 iCloud / Dropbox / git 处理）
- 富文本编辑器（Vim 才是想要的）
- 数据库、配置文件（约定优于配置）
